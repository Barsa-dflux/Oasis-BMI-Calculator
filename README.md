# 🏋️ BMI Calculator

## 📖 Description

This is a Python-based BMI (Body Mass Index) Calculator that helps users calculate their BMI using their weight and height. The program also provides BMI categories and health advice.

## ✨ Features

* ✅ Calculate BMI accurately
* 📊 Displays BMI category
* 💡 Provides health advice
* 👤 Takes user details such as name, age, and gender
* 💾 Option to save BMI report in a text file

## 🛠️ Technologies Used

* 🐍 Python

## 🚀 How to Run

1. Install Python on your system.
2. Download the project files.
3. Open the project in VS Code.
4. Run the `bmi_calculator.py` file.
5. Enter the required details when prompted.

## 📏 BMI Categories

* 🔹 Underweight: BMI less than 18.5
* 🟢 Normal Weight: BMI 18.5 – 24.9
* 🟠 Overweight: BMI 25 – 29.9
* 🔴 Obese: BMI 30 or above

## 📸 Project Output

The program displays:

* 👤 User information
* 📈 BMI value
* 📋 BMI category
* 💡 Health advice
